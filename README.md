# DJ Library

A fast DJ track library browser built for live use.

**Live:** https://akabuda050.github.io/djlibrary/

## Features

- Search by title or artist
- Filter by BPM range (preset buttons)
- Filter by key (toggle pills)
- Opens tracks on YouTube Music
- Works on mobile

## Stack

- Vue 3 + Vite
- Tailwind CSS v4
